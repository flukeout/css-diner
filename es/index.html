CSS Diner

Sin preocupaciones, ¡tú puedes hacer esto!

¡Estás a punto de aprender Selectores CSS! Los Selectores son cómo eliges a qué elemento aplicar los estilos.

Exhibición 1 - Una regla CSS

p {
   margin-bottom: 12px;
}

Aquí, el "p" es un selector (selecciona todos los elementos <p>) y aplica el estilo de margin-bottom.

Para jugar, tipea el selector CSS en el editor de abajo para seleccionar los items correctos en la mesa. Si eliges correctamente, avanzarás al siguiente nivel.

Pon el ratón sobre los elementos en la mesa para saber su estructura HTML.

¡Obtén ayuda con los selectores en la derecha! →
